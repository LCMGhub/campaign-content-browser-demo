    How to use this app

    Browse campaign email and landing page content for one client at a time. Work through the filters on the left; the preview updates on the right. This public build uses synthetic mock data only — no Domo connection.

    Steps

    1. Select a client — Required first. Loads that partition’s content list and filter options.
    2. Academic years (optional) — Click one or more year chips to narrow the list. Click again to deselect. Leave empty for all years.
    3. Program solutions (optional) — Click one or more chips to filter by program solution. Leave empty for all.
    4. Asset type — Switch between Emails and Landing Pages.
    5. Content name — Search by title, then click a row to preview. Use the pager if there are many results.
    6. Preview — Rendered on the right. For emails, header lines (Subject, From, Pre-Header) appear above the HTML body.

    Other controls

    • Reset — Clears the client and all filters so you can start over.
    • Hide filters / Show filters — Collapses the sidebar to give more room to the preview.
    • Truncation banner — If shown, the dataset hit the row limit; narrow academic year or program solution filters to load a smaller set.
